# Personal OS

A clean, interactive personal productivity dashboard built with React + Vite.

## Features

- **Today** – Daily tasks with carry-forward logic
- **Schedule** – Weekly planner with day-by-day view
- **Projects** – Workspace with links, notes, and checklists
- **Resources** – Knowledge base with inline links and tags
- **Monthly** – Progress tracking with subtask goals
- **Goals** – Long-term goals with deadlines and action lists

## Run locally

```bash
npm install
npm run dev
```

All data is stored in `localStorage` — no backend needed.
